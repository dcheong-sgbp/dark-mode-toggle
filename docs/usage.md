# Usage & API

It is a standard custom element, so it works with no wrapper in plain HTML, React, Vue, Svelte and Astro.

## Plain HTML

```html
<script src="dark-mode-toggle.js"></script>
<dark-mode-toggle></dark-mode-toggle>
```

## React

```jsx
import "@sgbp/dark-mode-toggle";
export default function Page() { return <dark-mode-toggle />; }
```

## Vue

```vue
<script setup>
import "@sgbp/dark-mode-toggle";
</script>

<template>
  <dark-mode-toggle />
</template>
```
